Overview

The UMass Cybersecurity Club organizes UMassCTF 2025, a global cybersecurity competition featuring challenges in web exploitation, cryptography, reverse engineering, and more. This event aims to provide an engaging learning experience through expert talks, networking opportunities, and exciting prizes.

Challenge 1: Steganography + Cryptography

This challenge presents an image packed with multiple QR codes, each possibly holding a piece of the puzzle.

Step 1: Image Analysis with Aperi’Solve

I uploaded the image to Aperi'Solve, which extracted hidden data including a suspicious hash and a key.

Step 2: Decrypting with Vigenère Cipher

Using DCode’s Vigenère Cipher Solver, I decrypted the hash with the key obtained earlier. This revealed the flag.

Challenge 2: PCAP Network Analysis

Step 1: Prepare the File

I first ran the following command to make the PCAP file usable:

chmod +x chall.pcapng

Step 2: Analyze in Wireshark

Loaded the file in Wireshark and used the filter:

frame contains "UMASS"

This revealed the flag inside a packet containing "UMASS".
